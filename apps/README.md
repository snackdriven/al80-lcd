# Apps

Vendor / third-party tools for working with the AL80. Kept here as version-pinned backups (the download pages change and old builds disappear). Not my work; original copyright applies.

| File | What it is | Size |
|------|-----------|------|
| `AL80_LCD_SCREEN_V1007_20241219.exe` | YUNZII's LCD screen app (V1007, dated 2024-12-19). The desktop uploader for images/GIFs to the LCD. The `research/site_assets/` JS bundle is the web version of the same thing. | 32 MB |
| `qmk_toolbox.exe` | QMK Toolbox — flashing/DFU utility. Only relevant if you ever restore firmware. Reminder: reflashing is what this project avoids; see `../firmware/`. | 8.6 MB |

Both are re-downloadable (YUNZII: yunzii.com/pages/software; QMK Toolbox: github.com/qmk/qmk_toolbox) — the point of keeping them here is having the exact builds that match this setup.
